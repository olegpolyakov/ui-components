# Flex

Flex is a layout component that arranges its children in a flexible box model, allowing for responsive and dynamic layouts.

<Demo
    setup={(props, setProps) => (
        <input
            type="number"
            min="1"
            value={props.count || 3}
            onChange={e => setProps({ count: Number(e.target.value) })}
        />
    )}
    settings={Flex.settings}
>
    {({count, ...props}) => (
        <Flex gap="s" {...props}>
            {Array.from({ length: count ?? 3 }).map((_, index) => (
                <div
                    key={index}
                    style={{
                        width: '100px',
                        height: '100px',
                        display: 'flex',
                        alignItems: 'center',
                        justifyContent: 'center',
                        backgroundColor: 'var(--kui-primary-color)',
                        color: 'var(--kui-on-primary-color)'
                    }}>
                    {index + 1}
                </div>
            ))}
        </Flex>
    )}
</Demo>
