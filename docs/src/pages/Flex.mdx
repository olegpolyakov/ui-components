# Flex

Flex is a layout component that arranges its children in a flexible box model, allowing for responsive and dynamic layouts.

<Demo settings={Flex.settings}>
    {({count, ...props}) => (
        <Flex gap="s" {...props}>
            {Array.from({ length: count }).map((_, index) => (
                <div
                    key={index}
                    style={{
                        width: '100px',
                        height: '100px',
                        display: 'flex',
                        alignItems: 'center',
                        justifyContent: 'center',
                        backgroundColor: 'var(--ui-primary-color)',
                        color: 'var(--ui-on-primary-color)'
                    }}>
                    {index + 1}
                </div>
            ))}
        </Flex>
    )}
</Demo>
